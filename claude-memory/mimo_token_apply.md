## 活动信息

- **活动**: 小米 MiMo Orbit 百万亿 Token 创造者激励计划
- **主办方**: 小米（Xiaomi MiMo）
- **总量**: 100 万亿 Token（免费发放）
- **时间**: 2026年4月28日 ~ 5月28日（30天，赠完即止）
- **申请地址**: https://100t.xiaomimimo.com
- **评估周期**: 约 3 个工作日
- **权益到账**: 收到邮件后 24h 内

## 申请步骤

1. 打开 `https://100t.xiaomimimo.com`
2. 点击「立即申请」按钮
3. 填写表单：
   - 邮箱（用于接收结果 + 登录开放平台）
   - AI 开发/Agent 工具（多选，如 Claude Code）
   - 底层模型系列（如 Claude 系列）
   - 项目描述（100词以上，写详细可提高通过率）
   - 证明上传（可选：账单截图/日志/GitHub链接）
4. 完成**滑块验证码**（需手动操作）
5. 点击提交

## 注意事项

- 申请邮箱需与 MiMo API 开放平台注册邮箱一致
- 填写越详细，评估通过率和权益档位越高
- 提交后约 3 个工作日收到邮件通知
- 若 3 天未收到邮件可重新提交
- 权益有有效期，过期作废
- 开放平台地址: https://platform.xiaomimimo.com
- MiMo 官网: https://mimo.xiaomi.com
